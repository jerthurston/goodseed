# GoodSeed

A cannabis seed price comparison and favoriting application.

## Overview

GoodSeed helps users compare cannabis seed prices from multiple dispensaries, save their favorite products, and get the best deals.

## Tech Stack

- **Framework**: Next.js 16.1.6 (App Router)
- **Database**: PostgreSQL with Prisma ORM
- **Cache/Queue**: Redis (Upstash) with Bull
- **Authentication**: NextAuth.js v5
- **Deployment**: Vercel (main app) + Render.com (worker)
- **Styling**: Tailwind CSS + shadcn/ui
